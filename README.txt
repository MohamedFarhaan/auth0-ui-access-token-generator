PRE-REQUISITES:
    http-server:
        run command 'npm install http-server -g'
--------------------------------------------------
Extract the Zip to a folder,
Replace values inside 'auth_config.json'
Open terminal in the folder's directory
Run Command 'http-server -a localhost -p 4200 -o'
Login Using Auth0 Credentials, and copy the access token
Use "CONTROL + C" to stop the server
